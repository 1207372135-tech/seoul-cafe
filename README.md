# Seoul Café & Books ☕

Proyecto desarrollado con **React** y **Vite** como parte de un taller práctico de desarrollo web.

## Descripción

Seoul Café & Books es una página web inspirada en una cafetería de estilo coreano ubicada en Ecuador. El sitio permite conocer el menú, la biblioteca, la galería y la información de contacto del establecimiento mediante una interfaz moderna y atractiva.

## Características

* Inicio (Hero)
* Barra de navegación (Navbar)
* Menú de bebidas y postres
* Sección Biblioteca
* Galería de imágenes
* Información de contacto
* Pie de página (Footer)
* Diseño responsivo

## Tecnologías utilizadas

* React
* Vite
* CSS3
* JavaScript

## Instalación

```bash
npm install
npm run dev
```

## Compilar el proyecto

```bash
npm run build
```

## Despliegue

```bash
npm run deploy
```
